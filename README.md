# Health Pilot

1v1 减脂健康管理 AI 教练 API，基于 Agno AgentOS。

## 功能特性

**1. 智能交互与多模态记录**
- **自然语言对话**：告别繁琐的表单填报，只需告诉 AI“我刚吃了一碗黄焖鸡米饭”，系统即可自动识别饮食意图并估算热量。
- **图片一键识别**：支持发送食物照片、外卖订单截图、体重秤照片、运动 App 截图，AI 将自动 OCR 并结构化提取关键数据（热量、三大营养素、运动时长、体重等）。

**2. 核心健康追踪**
- **饮食追踪**：智能估算食物的卡路里、蛋白质、碳水和脂肪，无需依赖死板的食物数据库。
- **体重与体脂追踪**：记录并分析体重变化趋势，支持体脂率记录。
- **运动记录**：记录不同类型的运动及消耗的热量，并与基础代谢（TDEE）结合计算当日热量缺口。

**3. 数据分析与可视化总结**
- **多维度报表**：提供每日、每周、每月的健康数据摘要，让你对进度了如指掌。
- **目标完成度追踪**：直观展示体重变化趋势以及蛋白质等核心营养素的达标率。

**4. Agentic Memory 长期记忆**
- **个性化档案**：初次使用时引导建立身体数据档案（身高、体重、目标、TDEE），并在后续交互中持续更新。
- **习惯与偏好学习**：随着对话深入，AI 会记住你的饮食偏好和行为模式，给出越来越懂你的建议。

**5. 主动关怀与智能推送闭环**
- **定时贴心提醒**：支持自定义早/中/晚餐打卡提醒、定期称重提醒和周报总结推送。
- **实时行为触发预警**：
  - *热量预警*：当记录的饮食导致当日摄入接近 TDEE 上限时，立即触发提醒。
  - *营养补足*：晚餐时段若蛋白质严重不达标，主动建议高蛋白菜品。
  - *达标鼓励*：连续多天热量控制在目标范围内，触发特别鼓励。
  - *平台期破局*：检测到体重连续多日无明显变化，主动提供调整饮食或运动的建议。
- **沉默用户唤醒**：当检测到你连续 24 小时未记录数据时，AI 会主动发来温暖的问候。
- **上下文记忆闭环**：当 AI 主动给你发了推送，你回复它时，它能“记起”刚才说过的话，顺着语境自然交流，就像真正的贴身私教。

## 技术栈

- Python 3.13+ / uv
- Agno AgentOS（FastAPI）
- 阿里云百炼 qwen3.5-plus（多模态，OpenAI 兼容）
- SQLAlchemy + SQLite（开发）/ PostgreSQL（生产）
- APScheduler（定时推送）

## 架构与技术细节

> 💡 **详细的 Agent 提示词设计与实现机制，请参考 [AGENT_DESIGN.md](./AGENT_DESIGN.md)**

- **单 Agent + 多 Tools 架构**：核心基于 `Coach Agent`，辅以饮食记录、体重记录、数据分析等专门的 Tools，实现意图识别与任务执行的统一。
- **动态 Prompt 注入**：Coach Agent 的 `instructions` 不是静态的，每次运行前会动态查询数据库，将用户的身体档案（身高/体重/TDEE）和今日实时数据（已摄入热量、营养素进度）注入上下文，使得 AI 的回复始终基于最新数据。
- **推送生成器 (Push Agent)**：独立于主对话 Agent，采用专门的精简 Prompt 根据定时任务或事件触发生成简短、温暖的推送文案，存入 `Notification` 数据库。
- **事件驱动的条件触发**：在用户使用 Tracker 工具（如记录一顿高热量的炸鸡）并完成数据库 `commit` 后，系统通过后台线程（`threading.Thread`）异步调用触发器检查，实现“数据一记录，预警即生成”的低延迟响应，且不阻塞当前对话。
- **对话记忆闭环 (Context Bridge)**：Coach Agent 每次组装 Prompt 时，会自动检索过去 2 小时内最近一条发给用户的未读主动推送。如果存在，即作为【系统上下文】告知 AI，从而让 AI 能够理解用户对于主动推送的后续回复，实现“系统找人 -> 人回系统 -> 系统接话”的完整闭环。

## 快速开始

```bash
# 安装依赖
uv sync

# 配置环境变量
cp .env.example .env
# 编辑 .env 填入 API Key

# 启动
uv run python main.py
```

服务启动后运行在 `http://localhost:7777`。

## 环境变量

```
LLM_MODEL=qwen3.5-plus
LLM_API_KEY=<your-api-key>
LLM_BASE_URL=https://dashscope.aliyuncs.com/compatible-mode/v1
```

## API

| 端点 | 说明 |
|------|------|
| `POST /agents/health-coach/runs` | 对话（支持文字+图片） |
| `GET /api/v1/notifications?user_id=xxx` | 查询推送通知 |
| `POST /api/v1/notifications/{id}/read` | 标记已读 |
| `GET /health` | 健康检查 |
| `GET /docs` | API 文档 |
